# 2025-06-25 工作日志

## 蜘蛛侠蛛网射手小游戏开发
- 开发了基于摄像头手势识别的蜘蛛侠蛛网射击网页游戏
- 文件：`/Users/wjs/Desktop/aiGame/spider-web-shooter.html`（单文件 HTML）
- 技术栈：MediaPipe Hands (tasks-vision) + Canvas 2D + Web Audio API
- 核心功能：
  - 摄像头授权 + 镜像显示
  - 蜘蛛侠手势识别（食指+小指伸、中指+无名指弯）
  - 蛛网射线动画（从手腕射出 → 延伸至屏幕边缘 → 蛛网喷射特效 → 淡出）
  - 粒子效果 + THWIP 文字 + 合成音效
  - 双手支持 + 冷却系统（450ms）
  - 手部骨骼可视化（H键切换）
- 2025-06-25 更新：
  - 移除开始按钮，页面加载后自动启动摄像头
  - 增加文件协议检测：file:// 打开时提示使用 HTTP 服务器
  - 音频改为延迟初始化：首次用户点击/按键后启用
  - 修复摄像头加载时机，使用 `loadeddata` 事件
  - 将 MediaPipe 方案从 tasks-vision (Google Cloud 模型) 切换为经典 MediaPipe Hands (CDN 加载库+模型)，并加入 elemecdn/jsdelivr/unpkg 多 CDN 回退
  - 增加 30 秒模型加载超时检测和对应中文错误提示
